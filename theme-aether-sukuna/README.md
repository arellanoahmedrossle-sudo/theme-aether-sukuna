# Aether Theme — Sukuna

Aether — Sukuna is a dark Visual Studio Code theme generated from a wallpaper using an automated palette extraction pipeline.

This edition is inspired by **Ryomen Sukuna**, blending deep charcoal backgrounds, crimson shadows, warm amber highlights, and muted gold syntax colors into a cohesive coding experience.

![Preview](preview.png)

## Features

- 🌑 Dark theme designed for low-light environments
- 🎨 Wallpaper-derived color palette
- ✨ Semantic syntax highlighting
- 🟥 Rich crimson accents balanced with warm gold highlights
- 👁️ High contrast without oversaturation

## Palette

| Role | Color |
|------|-------|
| Background | Charcoal Black |
| Primary Accent | Crimson Red |
| Secondary Accent | Warm Amber |
| Keywords | Scarlet |
| Functions | Muted Rose |
| Types | Soft Beige |
| Variables | Golden Yellow |

## Installation

### From VSIX

1. Open **Visual Studio Code**.
2. Open **Extensions** (`Ctrl + Shift + X`).
3. Click **⋯ → Install from VSIX...**
4. Select the generated `.vsix` file.

Then choose:

`Preferences → Color Theme → Aether`

## Development

The theme is automatically generated from wallpaper colors and packaged as a VS Code extension using `vsce`.
